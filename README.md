## 💡 نحوه استفاده

حالا اگه ربات مد نظرون رو در گروه عمومی ادمین کنید و کلمه #فعال رو در گروه ارسال کنید، ربات فعال مبشه و اگه عبارت (جوک، ضرب المثل، چالش، تاس، تاریخ) رو در گروه ارسال کنید، ربات یکی از جوک ها یا چالش ها و... رو که بهش دادیم رو در گروه میفرسته


| دستور | عملکرد |
|---|---|
| `#فعال` | فعال‌سازی ربات در گروه |
| `جوک` | ارسال یک جوک تصادفی |
| `ضرب المثل` | ارسال یک ضرب‌المثل |
| `چالش` | ارسال یک چالش |
| `تاس` | پرتاب تاس |
| `تاریخ` | نمایش تاریخ |
